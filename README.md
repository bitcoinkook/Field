# Field

*The third instrument of The Open Ring*

---

## What it is

Field reads real world outcomes from people who used The Gradient or SignalChain, acted on the output, and documented what happened.

It answers one question across all cases: is the framework predicting correctly?

---

## What it shows

- Total cycles completed
- Percentage that moved toward correct orientation
- Percentage that produced unexpected results
- Breakdown by tool used, domain experience, and outcome
- All individual field records, expandable

The unexpected results are as important as the correct ones. The framework is only valuable if it can be wrong and the data shows where.

---

## The Open Ring

Field is the third instrument in a three-tool feedback cycle:

```
The Gradient → SignalChain → Field → repeat
```

**The Gradient** identifies whether a structure's fixed and variable elements are correctly arranged and maps the opportunity.

**SignalChain** locates exactly where in the formula information stops flowing.

**Field** records what happened when you acted on that output in the real world.

The output of each tool becomes the input of the next. The cycle repeats with better data each time.

---

## Data source

Field reads from a public Google Sheet connected to the Open Ring field data form. The sheet ID is hardcoded. Responses are public and aggregated openly.

Submit a field record: [Open Ring Field Data Form]

---

## Important note

Field is only as useful as the honesty and domain experience of the people reporting. A cycle completed by someone with no real contact with the structure they analyzed produces noise, not signal.

The prerequisite for The Open Ring is genuine experience. The tools organize what you already know from direct contact. They do not replace that contact.

---

## Running it yourself

This tool reads from a public Google Sheet. No API key required. To use with your own data:

1. Clone this repository
2. Replace the `SHEET_ID` constant with your own Google Sheet ID
3. Deploy the React component to any hosting platform
